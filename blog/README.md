# Backend Engineer Take Home Test - NoLimit

## Tech Stack

- Java 21
- Spring Boot 3
- Spring Security
- JWT Authentication
- PostgreSQL
- Spring Data JPA
- Swagger OpenAPI

## Features

- User Registration
- User Login (JWT)
- Create Post
- Get All Posts
- Get Post by ID
- Update Own Post
- Delete Own Post

## API Documentation

Swagger UI:

http://localhost:8080/swagger-ui/index.html

## Database

PostgreSQL

## Run Project

1. Clone repository
2. Create PostgreSQL database
3. Configure `application.properties`
4. Run the Spring Boot application
5. Open Swagger UI
